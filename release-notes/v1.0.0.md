# v1.0.0 - Innioasis Y1 AirPods 2 No Sound Fix

This release describes the confirmed fix for AirPods 2 connecting to the Innioasis Y1 but producing no sound during music playback.

## Confirmed

- Confirmed on official firmware 3.0.7.
- Fixes the AirPods 2 connected-but-silent playback problem.
- Bluetooth works.
- Music playback works.
- AirPods media controls work:
  - play/pause
  - next track
  - previous track

## Final Layout

```text
/system/lib/libbluetoothdrv.so       = RTP timestamp fix proxy
/system/lib/libbluetoothdrv_real.so  = original official 3.0.7 libbluetoothdrv.so
/system/lib/libmtkbtextadpa2dp.so    = untouched original library
```

No separate AVRCP patch was needed.

## Firmware Policy

No firmware or vendor binaries are included. Users must build or prepare the patched image locally using their own legally obtained official firmware.
