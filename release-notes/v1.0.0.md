# v1.0.0 - Official 3.0.7 AirPods RTPFIX documentation

This release documents the confirmed Innioasis Y1 official firmware 3.0.7 AirPods RTP timestamp fix.

## Confirmed

- Working on official Innioasis Y1 firmware 3.0.7
- AirPods audio works
- AirPods controls work:
  - play/pause
  - next track
  - previous track
- No separate AVRCP patch is needed

## Firmware Policy

No firmware is included in this release. No vendor binaries, patched binaries, device dumps, or Bluetooth logs are included.

Users must legally obtain official firmware and build the patched image locally.
