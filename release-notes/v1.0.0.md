# v1.0.0 - Innioasis Y1 AirPods 2 No Sound Fix

This release documents the confirmed fix for AirPods 2 connecting to the Innioasis Y1 but producing no sound.

## Confirmed

- Confirmed on official firmware 3.0.7.
- Fixes AirPods 2 connected but no sound.
- Bluetooth works.
- Music playback works.
- AirPods media controls work:
  - play/pause
  - next track
  - previous track

## Final Layout

```text
/system/lib/libbluetoothdrv.so       = proxy
/system/lib/libbluetoothdrv_real.so  = original
/system/lib/libmtkbtextadpa2dp.so    = untouched
```

No separate AVRCP patch is needed.

## Firmware Policy

No firmware or vendor binaries are included. Users must build or prepare the patched image locally using their own legally obtained official firmware.
