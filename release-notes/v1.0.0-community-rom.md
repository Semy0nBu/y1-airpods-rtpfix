# v1.0.0 - Innioasis Y1 AirPods 2 Fix Community ROM

This is an unofficial community ROM for the Innioasis Y1 based on official firmware 3.0.7.

It fixes the problem where AirPods 2 connect over Bluetooth, but music playback is silent.

## Confirmed working

* Innioasis Y1 boots normally
* Bluetooth works
* AirPods 2 connect
* Music playback works
* Play/pause from AirPods works
* Next track from AirPods works
* Previous track from AirPods works
* USB storage / disk access works like stock firmware

## What was changed

Only the system image was changed.

The release build keeps normal stock-like USB storage behavior and does not enable ADB.

Final Bluetooth library layout:

```text
/system/lib/libbluetoothdrv.so       = RTP timestamp fix proxy
/system/lib/libbluetoothdrv_real.so  = original official 3.0.7 libbluetoothdrv.so
/system/lib/libmtkbtextadpa2dp.so    = untouched original library
```

## Important notes

This is not an official Innioasis firmware release.

This is an unofficial community build.

Use it only on Innioasis Y1.

Flash at your own risk.

Make sure the battery is charged before flashing.

Do not disconnect USB while flashing.

## Installation

1. Download the community ROM zip from this release.
2. Open the Innioasis Updater toolkit folder.
3. Use the SP Flash Tool helper.
4. Run:

```text
2. Run Me + Connect Y1
```

5. Fully power off the Y1.
6. Connect the Y1 by USB when the tool waits for the device.
7. Wait for flashing to finish.
8. Boot the player and test AirPods 2 playback.

## Rollback

If something goes wrong, flash the original Innioasis Y1 firmware 3.0.7 again using Innioasis Updater / SP Flash Tool helper.
